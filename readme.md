# Redstone  

## Описание

> Во всеми любимой игре MineCraft существует материал, позволяющий строить различные цепочки.  
> Конечно, название данного творения говорит само за себя, о каком материале идет речь.  
> Данный "язык программирования" является попыткой реализовать широкие возможности редстоуна с помощью кода.*

## Основная идея

Redstone в самой игре как только не использовали, и что только на нем не делали. Язык позволяет создавать схемы и использовать блоки, которые, к слову, можно самостоятельно написать.

- В нем отсутствуют if-else, for-while
- Нет определенной парадигмы
- Заставит вас страдать при попытке с нуля без модулей написать хоть что-то

## Синтаксис команд

- `input <имя> <0|1>` — установить входной сигнал (0 или 1)
- `output <имя>` — вывести значение сигнала
- `wire <откуда> <куда>` — передать сигнал по проводу
- `button <имя>` — нажать кнопку (установить сигнал)
- `torch <имя>` — активировать факел (сигнал = 1)
- `repeater <имя>` — повторить сигнал (усилить)
- `comparator <a> <b> <c>` — сравнить сигналы a и b, результат в c (1 если равны, иначе 0)
- `power <имя>` — включить питание

## Примеры

### Кнопка включает лампу

```plain
input button 1
button lamp
output lamp
```

### Передача сигнала через провод

```plain
input a 1
wire a b
output b
```

### Повторитель

```plain
input x 1
repeater x
output x
```

### Компаратор

```plain
input a 1
input b 1
comparator a b c
output c
```

### Факел

```plain
input lever 1
torch lamp
output lamp
```
